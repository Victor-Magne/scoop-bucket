# scoop-bucket

[Scoop](https://scoop.sh) bucket for [CloudTray](https://github.com/Victor-Magne/claudtray) — a Windows system-tray monitor for AI assistant usage quotas (Claude Code, GitHub Copilot, Codex, Antigravity).

## Usage

```powershell
scoop bucket add victor-magne https://github.com/Victor-Magne/scoop-bucket
scoop install cloudtray
```

## Update

```powershell
scoop update cloudtray
```
