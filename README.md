# scoop-bucket

[Scoop](https://scoop.sh) bucket for [ClaudTray](https://github.com/Victor-Magne/claudtray) — a Windows system-tray monitor for AI assistant usage quotas (Claude Code, GitHub Copilot, Codex, Antigravity, OpenRouter, Gemini, Ollama).

## Usage

```powershell
scoop bucket add victor-magne https://github.com/Victor-Magne/scoop-bucket
scoop install claudtray
```

## Update

```powershell
scoop update claudtray
```
